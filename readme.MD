# Investment Calculator

## About

Simple web application developed to start my studies in TypeScript.

The project calculates the estimated growth of an investment based on:

- Initial amount
- Monthly contribution
- Monthly return rate
- Investment period in months
- Currency format

## Technologies

- HTML
- CSS
- Bootstrap
- TypeScript
- DOM manipulation

## Features

- Investment form
- Input validation
- Monthly investment calculation
- Summary rendering
- Monthly evolution table
- Currency formatting
- Clear/reset button
